# 🏨 Hostel Management System

A Hostel Management System developed in **C++** using Object-Oriented Programming (OOP) concepts and File Handling. This project was created to practice C++ programming concepts such as classes, inheritance, file handling, and user authentication.

---

## Features

- Student Registration (with password validation)
- Student Login
- Admin Registration
- Admin Login
- Automatic Room Allocation
- Display All Students
- Search Student by Registration Number
- Vacate Room
- Display Vacated Student Details
- Student Logout
- Admin Logout

---

## Technologies Used

- C++

- GCC (g++)
- File Handling
